
Description
-----------
This module provides integration with the Payeezy payment gateway.
The integration methods offered by this module are:
 - Hosted gateway (offsite)
 - Onsite payment gateway


Installation
------------
Download the module from D.O and install it the usual way:
 - Place it in the /modules or /modules/contrib directory
 - Go to 'Extend' as an administrator, and
 - Enable the module


Dependencies
------------
Module requires no external dependencies.


Setup
-----
 - To get started with Payeezy, refer:
  https://developer.payeezy.com/faqs/how-do-i-get-started

 - For further sandbox links and demo account refer :
  https://support.payeezy.com/hc/en-us/articles/203730579
